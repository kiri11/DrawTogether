Описание каждого уровня состоит из файла с названием level.xml, версии 1.0 (Fifth Edition) в кодировке utf-8.
В качестве табуляции используется четыре пробела.
Корневым элементом является level с обязательным атрибутом id. 
id должны быть уникальными в пределах одной коробки и совпадать с окончанием названия папки.
Остальные параметры указываются как элементы, вложенные в корневой. 
Если значение параметра различается в локализованных версиях, 
создается несколько элементов с различными значениями атрибута lang по стандарту ISO 639-1 плюс суффикс страны.
