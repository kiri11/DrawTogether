Описание каждой коробки состоит из файла с названием box.xml, версии 1.0 (Fifth Edition) в кодировке utf-8.
В качестве табуляции используется четыре пробела.
Корневым элементом является box с обязательным атрибутом id. 
id должны быть уникальными в пределах одного режима игры и совпадать с окончанием названия папки.
Если значение параметра различается в локализованных версиях, 
создается несколько элементов с различными значениями атрибута lang по стандарту ISO 639-1 плюс суффикс страны.
